# NÚCLEO — Sistema Financiero Adaptable

Sistema de gestión financiera modular para cualquier negocio.

## Módulos incluidos
- **Núcleo:** Dashboard, Ingresos, Gastos, Flujo de caja, Balance, Reportes
- **Inventario** (activable)
- **Proyecciones de ventas** (activable)
- **Costos específicos** (activable)

## Instalación local

```bash
npm install
npm run dev
```

## Deploy en Vercel o Netlify

1. Sube esta carpeta a GitHub
2. Conecta el repositorio en vercel.com o netlify.com
3. Build command: `npm run build`
4. Output directory: `dist`
